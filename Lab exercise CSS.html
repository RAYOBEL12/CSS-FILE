What is CSS

CSS is a styling language that is you to give the look and feel of an HTML document.
The stylimg the HTML document using is done majorly in three ways:

  1. in line
  2. block styling
  3. external styling

It must noted that the styling is the most recommended since it is created as CSS file independent of the HTML file.
Therefore, changes can easily be made on the external styling.
